## 📌 Overview
This project predicts whether a customer is likely to churn (leave a service) using Machine Learning techniques.  
It helps businesses take proactive actions to retain customers and reduce revenue loss.

---

## 🎯 Problem Statement
Customer churn is a major challenge for subscription-based businesses.  
The goal is to build a model that identifies customers at risk of leaving.

---
